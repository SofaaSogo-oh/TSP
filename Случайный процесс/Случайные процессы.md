# Случайный процесс #определение
Случайный процесс (с.п.) --- семейство случайных величин 
$$\{\xi(t)\}_{t\in T}$$
индексированных параметром $t$, который пробегает множество $T$ , которое содержит
бесконечное множество значений.
Аналогично случаю случайных величин, для расчета вероятностей, ассоциированных
со случайным процессом, необходимо выполнение следующего условия . Для каждого
натурального $n$, для любого набора из $n$ $x$-в действительных чисел, для любых $n$ значений
параметра $r$ должна существовать вероятность
$$P(\xi(t_1)<x_1,\dots,\xi(t_n)<x_n)$$
* Сечение СП --- случайная величина в момент времени $t$ $x_{t}(\omega)$
* Реализация СП для события $\omega$ --- функция $x(t)=x_\omega(t)$
