# Связано с
[[Взаимная корреляционная функция]]
# Содержание
## Качественная связь между СП
* Внутри одного СП
	* Значения СП некоррелированные, если $R_x(t_1, t_2)=0$ #определение
* Два СП
	* СП $x^i, x^j$ некоррелированные, если $R_{ij}(t_1, t_2)=0$ #определение 
	* СП $x^i,x^j$ коррелированные, если $R_{ij}(t_1,t_2)\neq 0$ #определение
## Количественная связь между СП
#определение Нормированная автокорреляционная функция 
$$r_x(t_1,t_2)=\frac{R_x(t_1,t_2)}{\sqrt{D_x(t_1)\cdot D_x(t_2)}}$$
#определение Нормированная взаимная автокорреляционная функция
$$r_{ij}(t_1,t_2)=\frac{R_{ij}(t_1,t_2)}{\sqrt{D_{x^i}(t_1)\cdot D_{x^j}(t_2)}}$$
Замечание: $r_x(t_1,t_2),r_{ij}(t_1,t_2)\in[-1,1]$
При наличии линейной связи $r=\pm1$, при отсутствии $r=0$.
